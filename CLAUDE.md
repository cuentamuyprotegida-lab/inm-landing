# INM Landing (JARVIS)

## Que es
Landing page con asistente AI integrado (JARVIS). Usa AI SDK v6 + CSSOrb animation + Linear Look design.

## Stack
- Next.js (App Router)
- AI SDK v6 (streamText, useChat)
- Tailwind CSS
- TypeScript
- Vercel deployment

## Reglas
- AI responses deben usar AI Elements o MessageResponse
- Dark mode obligatorio
- Mobile responsive
- Espanol (Costa Rica)
